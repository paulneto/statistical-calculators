Significance Testing Cheat Sheet

1. What is Significance Testing?

Significance testing helps determine whether an observed result is unusual enough to suggest that something special is happening, rather than being due to random chance.

2. Key Terms

  • Null Hypothesis (H₀): The default assumption (e.g., "There is no effect" or "The coin is fair").
  • Alternative Hypothesis (H₁): What you are testing for (e.g., "There is an effect" or "The coin is biased").
  • Significance Level (α): The threshold for determining significance (commonly 0.05 or 5%).
  • P-Value: The probability of getting an outcome as extreme as your result, assuming H₀ is true.
  • Test Statistic: A number summarizing how far your data is from the null hypothesis (e.g., a z-score).

3. Types of Tests

  • One-Tailed Test: Checks for an effect in one direction (e.g., "Is the coin biased toward heads?").
  • Two-Tailed Test: Checks for effects in both directions (e.g., "Is the coin biased toward heads or tails?").

4. Steps in Significance Testing

  1. State the Hypotheses:
    • Null Hypothesis (H₀): What is considered the default (e.g., "The mean is 50").
    • Alternative Hypothesis (H₁): What you are testing for (e.g., "The mean is not 50").
  2. Set the Significance Level: Common choices: α = 0.05 (5%), α = 0.01 (1%).
  3. Calculate the Test Statistic: Use the formula for your specific test (e.g., z-score): 
    z = (Observed Value - Mean) / Standard Deviation
  4. Find the P-Value: Use statistical tables or software to determine the p-value from your test statistic.
  5. Make a Decision:
    • If p-value < α: Reject H₀ (result is significant).
    • If p-value ≥ α: Fail to reject H₀ (result is not significant).

5. Common Examples

  • Coin Flip Experiment:
    • H₀: The coin is fair (mean = 50 heads in 100 flips).
    • Observed: 80 heads.
    • Standard Deviation: SD = √(n × p × (1-p)) = 5.
    • Z-Score: z = (80 - 50) / 5 = 6.
    • Result: Lies in the right tail, significant at α = 0.05.

6. Tips

  • Understand the Hypotheses: Be clear on H₀ and H₁ before starting.
  • Know the Difference Between One- and Two-Tailed Tests: Use two-tailed unless you have a specific directional prediction.
  • Practice Interpreting P-Values: A p-value is not the probability that H₀ is true; it’s the probability of your result (or more extreme) assuming H₀ is true.
  • Visualize Results: Draw the normal curve to see where your result lies.

7. Useful Formulas and Choosing the Right Test

Common Statistical Tests and When to Use Them:

  • Z-Test:
    • Use when comparing a sample mean to a known population mean.
    • Assumes the population standard deviation is known.
    • Ideal for large sample sizes (n > 30).
  • T-Test:
    • Use when comparing means, and the population standard deviation is unknown.
    • Types:
      • One-Sample T-Test: Compare a sample mean to a known value.
      • Two-Sample T-Test: Compare means between two independent groups.
      • Paired T-Test: Compare means of paired or matched samples (e.g., before and after measurements).
  • Chi-Square Test:
    • Use for categorical data to test relationships or goodness-of-fit.
    • Types:
      • Goodness-of-Fit Test: Determine if observed proportions match expected proportions.
      • Test of Independence: Assess if two categorical variables are related.
  • ANOVA (Analysis of Variance):
    • Use when comparing means across more than two groups.
    • Assumes the data is approximately normally distributed and variances are equal.
  • Regression Analysis:
    • Use for predicting the relationship between variables (e.g., linear regression).
    • Dependent variable must be continuous.
  • Non-Parametric Tests:
    • Use when data doesn’t meet normality assumptions or has small sample sizes.
    • Examples:
      • Mann-Whitney U Test: Non-parametric alternative to the t-test.
      • Kruskal-Wallis Test: Non-parametric alternative to ANOVA.

Useful Formulas:

  • Z-Score: 
    z = (Observed Value - Mean) / Standard Deviation
  • P-Value: Found using z-tables or software (e.g., Python, R).

Significance testing is a tool to help make decisions about data. Always interpret results in the context of your experiment!
